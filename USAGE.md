# 📖 快速使用指南

## 🚀 启动应用

1. **启动服务器**
```bash
npm start
```

2. **访问网页**
- 打开浏览器访问：http://localhost:3001

## 💡 使用技巧

### 🎯 推荐的测试流程

1. **使用默认地址**：页面会自动填入SOL代币地址 `So11111111111111111111111111111111111111112`

2. **快速测试不同时间粒度**：
   - 点击 "10秒" - 查看超短期趋势（10个数据点）
   - 点击 "1分钟" - 查看短期波动（12个数据点，5秒间隔）
   - 点击 "1小时" - 查看中期趋势（12个数据点，5分钟间隔）

3. **分析建议理解**：
   - 🟢 **建议买入**：综合信心度 > 50%
   - 🔴 **建议卖出**：综合信心度 > 50%（看跌）
   - 🟡 **建议观望**：市场信号不明确

### 📊 图表说明

- **价格走势图**：显示代币价格变化趋势
- **资金流向图**：
  - 绿线：买入资金量
  - 红线：卖出资金量  
  - 橙线：净流入（买入-卖出）
- **交易用户图**：
  - 绿线：买家数量
  - 红线：卖家数量
  - 紫线：净用户差（买家-卖家）
